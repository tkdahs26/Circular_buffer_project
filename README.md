# Circular_buffer_project

Language: C++<br><br>

Tool: Visual Studio 2022<br><br>

효율적인 메모리 관리가 가능한 순환 버퍼를 구현하고, STL 알고리즘과 호환되는 반복자Iterator를 사용한 프로젝트입니다.<br><br>


<br>주요 기능
<br>템플릿클래스활용한 다양한 데이터 타입
<br>`head`와 `tail` 포인터를 성능의 데이터 삽입

<br><br>

프로그램 실행 시, 버퍼가 가득 찬 후 새로운 데이터(`26.1`)가 들어올 때 가장 오래된 데이터(`23.5`)를 덮어쓰는 동작을 확인할 수 있습니다.
